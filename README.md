# Farm Simulation (JavaFX)

Academic simulation project in Java/JavaFX: grid-based farm, entities (plants/animals), resources and seasonal cycle.

## Highlights
- 6x6 grid simulation
- Plants & animals with resource constraints
- Shop + warehouse stats (JavaFX UI)
- Seasonal logic and end-of-season selling

## Tech
Java • JavaFX • OOP • Simulation

## Structure
- `fermeAuto/src/` : source code
- `fermeAuto/data/` : images + configuration
- `diagrammeDeClasse.mermaid` : class diagram

## Compile & Run (JavaFX)

Main entry: package `MainPrincipale`.

### Requirements
- Java (JDK 17 recommended)
- JavaFX SDK

### VM options (JavaFX)
Set the following VM options when running the project:

```text
--module-path /path/to/javafx-sdk/lib --add-modules javafx.controls,javafx.fxml

### Quick start (IDE)
1. Import the project in your IDE
2. Configure JavaFX SDK
3. Run the main class from package `MainPrincipale`
4. Add the VM options above
